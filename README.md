# C++ Array Input and Summation

A beginner-friendly C++ console application that demonstrates how to work with arrays, constant variables, and loops.

## 💡 How it Works
The program initializes an array of size 15. It then performs two main tasks:
1. **Input Phase:** Prompts the user to enter 15 separate integers.
2. **Processing Phase:** Iterates through the array to print the numbers and calculate the cumulative sum.



## 🚀 Getting Started

### Prerequisites
You will need a C++ compiler installed on your machine (e.g., GCC, Clang, or MSVC).

### Execution
1. Clone the repository:
   `git clone https://github.com/YOUR_USERNAME/cpp-array-summation.git`
2. Compile the code:
   `g++ main.cpp -o sum_program`
3. Run the application:
   `./sum_program`

## 🛠️ Code Snippet
The core logic uses a standard `for` loop for summation:
$sum = \sum_{i=0}^{n-1} numbers[i]$
